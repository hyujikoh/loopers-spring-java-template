###  상품
- 상품은 고유한 상품 ID, 이름, 설명, 가격, 재고 수량을 포함해야 합니다.
- 가격은 0 이상이어야 한다.
- 재고 수량은 음수가 될 수 없습니다.
- 하나의 상품은 하나의 브랜드에 속한다.
- 상품은 복수의 상품을 조회할수있고 없을 경우엔  404 에러를 반환해야한다.
- 상품은 상세 조회가 가능하고 없을 경우엔 404 에러를 반환해야한다.
- 하나의 상품은 카테고리에 속한다. 

### 브랜드
- 브랜드는 고유한 브랜드 ID, 이름, 설명을 포함해야 합니다.
- 하나의 브랜드는 0개 이상의 상품을 외래키로 가져야한다.
- 브랜드는 복수의 브랜드를 조회할수있고 없을 경우엔  404 에러를 반환해야한다.
- 브랜드는 상세 조회가 가능하고 없을 경우엔 404 에러를 반환해야한다.
